Secure Telethon Session Guide

The safe way to generate your session string locally.

CRITICAL SECURITY WARNING

Never use any public website or online tool that asks for your phone number, API keys, or OTP to generate a session string. These websites can and will steal your Telegram account. The only secure method is to generate the string on your own computer, where your credentials never leave your control.

How to Generate Your Session String Safely

Step 1: Install Python

If you don't have Python installed, download and install it from the official website. Make sure to check the box that says "Add Python to PATH" during installation.

Download Python

Step 2: Get Your Telegram API Credentials

You need to get your personal api_id and api_hash from Telegram.

  1. Log in to my.telegram.org.
  2. Go to "API development tools".
  3. Fill out the form (you can use any app name/short name).
  4. Your api_id and api_hash will be displayed. Keep them secret and safe.

Step 3: Create and Run a Python Script

Create a new file named get_session.py on your computer and paste the following code into it.


from telethon.sync import TelegramClient
from telethon.sessions import StringSession

print("Starting the session string generator...")

# Your API credentials from my.telegram.org
# It is recommended to enter them when prompted for security.
API_ID = input("Please enter your API ID: ")
API_HASH = input("Please enter your API HASH: ")

# The script will create a .session file in the same directory.
# We then print the session string from it.
with TelegramClient(StringSession(), API_ID, API_HASH) as client:
    session_string = client.session.save()
    print("\nSUCCESS! Your Telethon session string is:\n")
    print(f"{session_string}\n")
    print("Please COPY and SAVE this string securely. Do not share it with anyone.")

print("Process finished.")

Step 4: Run the Script from Your Terminal

First, you need to install the Telethon library. Open your terminal or command prompt and run:

pip install telethon

Now, navigate to the directory where you saved get_session.py and run the script:

python get_session.py

The script will ask for your API ID, API Hash, phone number, password (if you have one), and the OTP sent to you by Telegram. After you log in successfully, it will print your session string in the terminal.
